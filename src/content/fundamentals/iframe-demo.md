## Technical Notes

To embed an iframe in your article, simply paste the HTML iframe code directly into your markdown file. The system supports:

- YouTube videos (with automatic responsive wrapper)
- Google Forms
- Other iframe-embeddable content

**Example syntax:**
```html
<iframe src="YOUR_URL" width="100%" height="500"></iframe>
```

The platform will render these iframes safely while maintaining the security and integrity of the site.
